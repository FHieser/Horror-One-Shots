# Load Base Command

Load the three base content files into context before starting work on the one-shot.

## Files to read

Read all three files in parallel:

- `Close to God/webapp/header.md` — Overview page content (Background, Teaser, Introduction)
- `Close to God/webapp/story.md` — Story page (character creation, plot points)
- `Close to God/webapp/mechanics.md` — Mechanics page (all rules and systems)

## After reading

Confirm to the user that all three files are loaded, then give a short summary of each:
- Overview: one line on the current background/teaser state
- Story: one line on what's written vs. still placeholder
- Mechanics: list the section headings currently in the file

Then ask: **"What would you like to work on?"**
